# eMetalWorks - Steel Fabrication Calculator & Business Management

A modern, responsive web application for **Bhavya Fabrication Works** providing steel fabrication services with an intelligent calculator system and business management tools.

## 🚀 Features

### 🧮 **Dynamic Calculator System**
- **Smart Weight Factors**: Automatically adjusts based on grill type, rod thickness, spacing, and design complexity
- **Multiple Grill Types**: Window grills, security grills, decorative grills, balcony railings
- **Rod Specifications**: 8mm, 10mm, 12mm square rods with accurate weight calculations
- **Real-time Pricing**: Instant cost estimates with transparent calculations
- **Professional Accuracy**: Expert-verified weight factor of 1.3 kg/sq.ft for window grills

### 🎨 **User Experience**
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Tabbed Navigation**: Home, Services, Calculator, Portfolio, Contact
- **Smooth Animations**: Framer Motion powered transitions
- **Professional UI**: Modern design with Tailwind CSS

### 📊 **Business Management**
- **Contact Form Integration**: Calculator data included in submissions
- **Admin Dashboard**: View and manage customer inquiries
- **Lead Tracking**: Detailed specifications and requirements
- **Analytics**: Track calculator usage and customer interactions

## 🛠️ Tech Stack

**Frontend:** React 18 + Vite + Tailwind CSS + Framer Motion + React Router DOM
**Backend:** Node.js + Express.js + CORS + In-memory data storage
**Calculations:** Custom JavaScript engine with dynamic weight factors
**UI/UX:** Responsive design with professional animations
**Features:** Calculator, contact management, analytics, portfolio showcase

## 📦 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Start backend server (separate terminal)
cd server && npm start

# Access application
# Frontend: http://localhost:3000
# Admin: http://localhost:3000/admin (admin/admin123)
```

## 🧮 Calculator Features

### Dynamic Weight Factors
- **Window Grills**: 1.3 kg/sq.ft base factor
- **Security Grills**: 2.0 kg/sq.ft (heavy-duty)
- **Decorative Grills**: 1.2 kg/sq.ft (artistic)
- **Balcony Railings**: 2.8 kg/sq.ft (structural)

### Rod Specifications
- **8×8mm Square Rods**: Standard for window grills
- **10×10mm Square Rods**: +40% weight for security
- **12×12mm Square Rods**: +80% weight for heavy-duty
